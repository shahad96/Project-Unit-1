# Individual Elephants 

Individual elephants is a website to display information about famous elephants in history. All the information has been taking from ‘The Elephant’ API.

## Description

You can navigate the information in two ways:
* Navigate through the century (the century that the elephant born in).
* Navigate via the elephant specie.
when user chose specific elephant more details will show in popup, you can add comments too.

## Executing program

The html file index.html
